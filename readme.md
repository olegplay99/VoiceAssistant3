## 🎯 **ЧТО ЭТО?**

**Голосовой Командир** - это приколюха, которая превращает твой голос в магию!

Ты просто говоришь, а комп делает всё, что скажешь:
- 🖥️ **Открывает любые программы** (даже игры!)
- 🌐 **Открывает популярные сайты**
- 📸 **Делает скриншоты**
- ⏰ **Говорит время и дату**
- 😂 **Прикалывается** (режим картавости, зубы, мемы)
- 💀 **Даже выключает компьютер!**

## 📋 **ПОЛНЫЙ СПИСОК КОМАНД**

### 🎯 **Основные команды**
| Команда | Что делает |
|---------|------------|
| `активация` / `активируйся` | Включает голосовое управление |
| `выключись` / `отключись` | Выключает ассистента |
| `время` / `сколько времени` | Говорит текущее время |
| `день` / `дата` / `число` | Говорит текущую дату |
| `скриншот` / `скрин` | Делает скриншот экрана |
| `я картавый` / `я шепелявый` | Включает режим картавости |
| `зубы` / `покажи зубы` | Смайлик показывает зубы |
| `игры` / `какие игры` | Показывает список всех игр |
| `громче` | Увеличивает громкость |
| `тише` | Уменьшает громкость |
| `выключи компьютер` | Выключает ПК через 10 секунд |
