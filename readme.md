# 🎙️ Голосовой Командир v0.43 BETA

**Голосовой помощник с ИИ, который шарит за мемы и всегда поможет!**

Управляй компом голосом как настоящий хакер! 😎

---

## 🎯 **ЧТО ЭТО?**

**Голосовой Командир** - это приколюха, которая превращает твой голос в магию!

Ты просто говоришь, а комп делает всё, что скажешь:
- 🖥️ **Открывает любые программы** (даже игры!)
- 📸 **Делает скриншоты**
- ⏰ **Говорит время и дату**
- 😂 **Прикалывается** (режим картавости, зубы, мемы)
- 💀 **Даже выключает компьютер!**

---

## 🔥 **КЛЮЧЕВЫЕ ФИЧИ**

### 🎤 **Голосовое управление**
Распознаёт русский язык через **Whisper** и выполняет команды!

### 😊 **Живой смайлик**
В углу экрана живёт смайлик, который:
- 🗣️ **Говорит** вместе с тобой
- 😱 **Реагирует** на клики (пощёлкай по нему!)
- 🦷 **Показывает зубы** по команде
- 😢 **Грустит**, если обидеть

### 🗣️ **Режим картавости**
Включается командой "я картавый" и ассистент начинает говорить с картавостью! 😂

### 🏴‍☠️ **Пиратский режим**
Команда "я пират!" - предупреждает, что тебя поймают!

### 📸 **Мгновенный скриншот**
Просто скажи "скриншот" и всё сохранится в папку!

### 🔧 **Настройка путей**
Меняй пути к программам прямо из трея!

---

## 📋 **ПОЛНЫЙ СПИСОК КОМАНД**

| Команда | Действие | Пример |
|---------|----------|--------|
| `открой [программа]` | Открывает программу | "открой хром" |
| `скриншот` | Делает скриншот | "скриншот" |
| `время` | Говорит текущее время | "сколько времени" |
| `день` | Говорит текущую дату | "какой сегодня день" |
| `зубы` | Смайлик показывает зубы | "покажи зубы" |
| `я картавый` | Включает режим картавости | "я картавый" |
| `я пират` | Включает пиратский режим | "я пират!" |
| `выключи комп` | Выключает ПК через 10 секунд | "выключи компьютер" |
| `активация` | Активирует голосовое управление | "активация" |
| `отключись` | Отключает ассистента | "выключись" |
