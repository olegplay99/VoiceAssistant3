## 🎯 **Что это?**

**Голосовой Командир** — твой личный голосовой помощник для Windows. Он всегда слушает (но не подглядывает) и активируется по слову **"компьютер"**. После этого ты можешь отдавать команды голосом:

- 🎵 **Управлять музыкой** (пауза, дальше, назад, громче/тише)
- 🌐 **Открывать сайты** (YouTube, TikTok, VK, ChatGPT и 25+ других)
- 🎮 **Запускать игры Steam** (CS2, Dota 2, GTA V и 20+ игр)
- 🖥️ **Открывать 150+ программ** (Adobe, Blender, OBS, продюсерские, кодерские)
- 📸 **Делать скриншоты**
- ⏰ **Говорить время и дату**
- 💀 **Выключать компьютер**
- 😂 **Прикалываться** (режим картавости, зубы, пиратский режим)

## 🔥 **Что нового в v1.0**

### 🎤 **Голосовая активация!**
Теперь **не нужно нажимать кнопку**! Просто скажи **"компьютер"** и ассистент проснётся!

- 🧠 **VAD (Voice Activity Detection)** — постоянно слушает микрофон
- 🎯 **Распознаёт только слово "компьютер"** — не грузит систему
- ⚡ **Мгновенная активация** — без задержек

### 😊 **Живой смайлик**
В углу экрана живёт смайлик, который:
- Говорит вместе с тобой
- Реагирует на клики (пощёлкай!)
- Показывает зубы по команде **"зубы"**
- Грустит, если обидеть

### 🗣️ **Режим картавости**
Скажи **"я картавый"** — и ассистент начнёт говорить с картавостью 😂

### 📸 **Мгновенный скриншот**
Просто скажи **"скриншот"** — файл сохранится в папку с программой
