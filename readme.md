## 🎯 **ЧТО ЭТО?**

**Голосовой Командир** — твой голосовой помощник, который открывает программы, сайты, игры и **теперь управляет музыкой!**

Ты просто говоришь, а комп делает всё:
- 🎵 **Управляет музыкой** (пауза, дальше, назад, громче/тише)
- 🌐 **Открывает сайты** (YouTube, TikTok, VK и 25+ других)
- 🎮 **Запускает игры Steam** (CS2, Dota 2, GTA V и 20+ игр)
- 🖥️ **Открывает 100+ программ** (Adobe, браузеры, игры)
- 📸 **Делает скриншоты** по голосу
- ⏰ **Говорит время и дату**
- 💀 **Выключает компьютер** голосом
- 😂 **Прикалывается** (режим картавости, зубы, смайлик)

## 🎵 **НОВОЕ В v0.84:**

- 🎵 **Управление музыкой голосом!**
  - "пауза" / "стоп" — остановить трек
  - "дальше" / "следующий" — следующий трек
  - "назад" / "предыдущий" — предыдущий трек
  - "громче" / "тише" — регулировка громкости
  - "включи музыку" — включить/выключить

- 🔧 **Улучшенное распознавание** — работает чётче и быстрее

- ⚡ **Оптимизация** — меньше жрёт ресурсов
